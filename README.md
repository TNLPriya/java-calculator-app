# Java Calculator

Simple calculator program using Java.

## Features
- Add
- Subtract
- Multiply
- Divide

## Concepts
- Classes and methods
- Loop
- User input using Scanner
## How to Run
1. compile:
  javac MainApp.java
2. Run:
   java MainApp
## Files
- Calculator.java - contains calcution methods
- MainApp.java - main program with menu
